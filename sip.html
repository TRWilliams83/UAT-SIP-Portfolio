SIP: Project C2 — Capillary Canopy

Design Prototype • SIP311 & 408 • Tests • Data

Innovation Claim

Claim: This project is innovative because it uses 3D-printed lid geometry. Not external equipment to create repeatable, tunable microclimates in small tanks, combining condensation routing, optional vent control, and an integrated reservoir + wick tube into a single modular lid system manufactured on a Bambu P1P from Fusion 360 models.

Project Description

This project designs and prototypes three 3D-printed lids for three 2.5-gallon glass tanks, modeled in Fusion 360 and printed on a Bambu P1P. Each lid will be a tuned variant (ex: arid / temperate / humid behavior) using the same core architecture: a sealed rim interface, controlled vent options, and a reservoir-fed wick tube that supports passive moisture movement. The lids will incorporate surface features—like internal slopes, drip rails, and condensation channels—to intentionally route condensed water (ideally away from the front viewing pane) and reduce user maintenance. The intended users are hobbyists, students, and “I want a tiny ecosystem, but I also want my weekends back” people who want a stable micro-environment without external pumps or constant attention.
